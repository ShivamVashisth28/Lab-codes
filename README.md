# Lab-codes

## Sorting file contains 
 1. Bubble Sort 
    - i: 0 to n, j: 0 to n-i-1
 2. Insertion Sort
    - i: 0 to n
    - j=i-1 and key = arr[i] , 
    - arr[j+1] = arr[j] and arr[j] > key
 3. Selection Sort 
    - i: 0 to n-1, j: i+1 to n, 
    - minIndex and swap minIndex with i
 4. Bucket Sort 
    - first form buckets
    - sort those buckets
    - combine orderly
